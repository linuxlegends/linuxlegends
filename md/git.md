# GIT

Exibe o status atual do projeto

```plaintext
git status
```

Prepara o arquivo.txt para commit

```plaintext
git add arquivo.txt
```

Prepara todos os arquivos presentes na pasta para commit

```plaintext
git add .
```

Faz o commit e sempre dÃª um nome do que foi alterado

```plaintext
git commit -m "nome do commit"
```

Finaliza o commit

```plaintext
git push
```

Fecha o terminal

```plaintext
exit
```
